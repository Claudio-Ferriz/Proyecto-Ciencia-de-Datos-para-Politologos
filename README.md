# Proyecto Ciencia de Datos para Politólogos

Proyecto para generar un espacio en el que los estudiantes puedan consultar información sobre las distintas materias, cátedras y comisiones de la carrera Ciencia Política de la Universidad de Buenos Aires. Junto a la consulta de evaluaciones, contenidos y opiniones generales de las distintas materias que componen la licenciatura.
